About Database

The database I chose was an IMDB movie and tv show database from the rapidapi site. The API was created by the OctopusTeam. It has a search feature that allows you to search for movies by title. The search feature is case insesitive so to make it easier to search for movies. It also has a feature where you can see the number of different genres, languages, countries, and entertainment types that can be found in the database. I used that information for the stats page. The database has a lot of information about each movie such as title, year, genre, language, country, release date, runtime, plot description, image, and the content rating.
