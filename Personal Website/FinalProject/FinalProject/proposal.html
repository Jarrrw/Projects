Chosen Base App

We selected the Simple Display Calender from the textbook. It includes features such as picking a month and year from 1990 to 2020 and returning the correct calender from that montha and year.

We are proposing an original app: Calender App. It is an app that will act as a planner with features similar in scope to the textbook examples.

Project Overview

Our application will allow users to add events to the calender, click on the event to view more info, and if they click on the on a specific day the add event form will appear with date already inputted

Proposed Enhancements

In addition to implementing the base app, we plan to make the following improvements and changes:

  • HTML: New form to enter events and display the event on the calender and display view more info
  • CSS: Style the view more info blocks, the form that user enters events, and how the events will display in the calender
  • JavaScript: Form Validation and getting the information for the calender
  • PHP: Calling the databases that hold the events and the user account information
  • Database: Search Functionality, Account Tables, Events Table
  • Authentication: Login/Logout with password hashing and optional user roles (e.g., user/admin)
